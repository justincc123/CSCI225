Portfolio

  1. RNG Game

    For this project we created an open world wizard game. With a group of 4 working the game was completed in six weeks. The game was created using java in Eclipse. The game is a text based RPG where the player can explore the world, fight enemies, and level up their character. The game featured many side quests and a main quest. There was a currency based shop where the player could buy weapons, armor, and potions. Using the save feature players could take a break and return at any moment.

  2. Budget Tracker

    For this project I used VB.NET in Microsoft Visual Studio. The tracker included multiple features the most outstanding one being a future teller. The tracker was able to hold any information the user inputted and display it in a readable format. The user could input their income and expenses so the tracker could display them or even calculate their future spendings.

  3. Plinko Ball

    For this project I used C++ in Microsoft Visual Studio. Over the course of 2 weeks I created a program that would mimic the famous gambling game Plinko. The program would drop a ball from the top of the board and wait for it to drop. The program included multiple features such as changing the price of the Plinko or the number of Plinkos dropped.

  4. 3D Poker Animation

    For this project I created a short animation that showed a game of poker being played. I used C++ in Microsoft Visual Studio. This was for a final project and used many different properties to complete. The most important property was light. The project focused on light reflection when being moved. The reflection was unique for each texture.

  5. 8 Ball Game

    For this project I used C++ in Microsoft Visual Studio. I created a fully functioning pool game. This program has heavy involvment of physics. The most difficult part was mimicing the table fabric slowing the ball and affecting how the balls bounced off the wall or each other.

  6. Air Hockey Game

    For this project I used C++ in Microsoft Visual Studio. For this I used a simple animation of a table. Using this animation I created a simple demo air hockey game. I added effects onto each puck, mallets, walls, and table top. With these effects the game mimiced all physics in an air hocvkey game.
